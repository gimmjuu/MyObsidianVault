## 📌 Linux에서 wget으로 URL 파일 다운로드
➰　`wget` `[옵션]` `[URL]`

1. **현재 디렉토리에 파일 다운로드**
```
$ wget https://example.com/file.zip
```
2. **파일명 지정하기**
```
$ wget -O output_file.zip https://example.com/file.zip
```
3. **저장 경로 지정하기**
```
$ wget -P /path/to/save https://example.com/file.zip
```
